Python စလေ့လာနေတာလား? Analytics အတွက်လား?

Python စလေ့လာချင်သူတွေအတွက် စလေ့လာနိုင်မယ့် နေရာတွေကို ဒီနေရာမှာ စုစည်းထားပါတယ်။ ကိုယ့်ကိုယ် google မှာ‌ ခေါက်ကြည့်လိုက်ရင်လည်း ဒီထက်မက ထွက်လာမှာပါ။ ကိုယ့်လို Python ကို ခုမှ စလုံးရေစ လေ့လာမယ်၊ အတွက်အချက်လည်း မကောင်းဘူး၊ ရိုးရိုးရှင်းရှင်းမှ သင်ချင်တယ်၊ ကိုယ့်ကိုယ်ပဲ လေ့လာမယ် ဆိုသူတွေအတွက် အထောက်အကူ အလို့ငှာ ရည်ရွယ်ပါတယ်။ စလေ့လာသူ မဟုတ်ဘူးဆိုရင်တော့ ဒီစာမျက်နှာနဲ့ မသက်ဆိုင်တော့ပါဘူး။

ကိုယ်လည်း အခုမှစတာဆိုတော့ ဒီစာမျက်နှာမှာ ပေးထားတာတွေ အကုန်လုံးက စလေ့လာသူနဲ့ အဆင်ပြေမယ်လို့ထင်ပါတယ်။ မပြေဘူးဆိုလည်း ကိစ္စမရှိပါဘူး။ google မှာ အချိန်ပေး ရှာကြည့်လိုက်ရင် ထွက်လာမှာပါ။ coding မသင်ဖူးဘူး မရေးဖူးဘူး ဆိုရင်လည်း စိတ်ပူစရာမလိုပါဘူး။

Python ကို လေ့လာဖို့ စိတ်ကူးမိတဲ့ အကြောင်းရင်းထဲမှာ သူက open source ဖြစ်တာရယ် သင်ယူဖို့ အလွယ်ဆုံး programming language ထဲက တခုဖြစ်တယ် ဆိုလို့ပါပဲ။ အဲ့လိုမှ မဟုတ်ရင်လည်း ယောင်ပြီးတောင် စမ်းမကြည့်ချင်ပါဘူး။ နောက်.. open source community ဟာ အရမ်းဖေးမကြပါသတဲ့။ ဟုတ်မဟုတ်တော့ ကိုယ့်ကိုယ်ပဲ ရှာဖွေဆန်းစစ်ကြည့်ကြပါ။ (Empathy is contagious. Compassion is contagious. စာနာစိတ်၊ ကရုဏာစိတ် ဆိုတာ ကူးစက်ရာဂါလို ကူးစက်တယ်လို့ သုတေသနတွေက ဆိုကြတာ သူတို့ကနေ မင်းဆီ ကူးစက်သွားပြီ မဟုတ်လား?) ဘယ်လိုဖြစ်ဖြစ်.. တခုခုသင်ဖို့ အထောက်အပံ့ အလကားရတယ်ဆိုတာလည်း အကောင်းဆုံးကိစ္စပဲ မဟုတ်လား?🙂

(နောက်ထပ် analytics အတွက် အသုံးများတဲ့ open source ကတော့ R ပါ။ academic မှာပဲ အသုံးများပါတယ်တဲ့။ Economics လို Business လို ဘာသာရပ်မျိုးကို နိုင်ငံခြားတက္ကသိုလ်တက်မယ့် သူတွေကတော့ ၂ မျိုးလုံး သင်ကြရမယ်ထင်ပါတယ်။)

ဒီစာမျက်နှာကိုဖြစ်ဖြစ် Python လေ့လာတာနဲ့ ပတ်သက်လို့ဖြစ်ဖြစ် အကြံပြုလိုတာ မှတ်ချက်ပေးလိုတာ ရှိရင် ဒီအီးမေးလ် 📩ကနေ ဆက်သွယ်လို့ရပါတယ်။

PY4E

ဒေါက်တာ ချက်ခ် ကိုတော့ ကျေးဇူးတင်ရမယ်။ coding မသင်ဖူးလဲ ကိုယ့်ကိုယ်လေ့လာဖို့ ok တယ်လို့ ဘာလို့ ပြောလဲဆိုတာ သူစာသင်တဲ့ ဗီဒီယို ကိုကြည့်ရင် သိပါလိမ့်မယ်။ အရိုရှင်းဆုံးဖြစ်အောင် စိတ်ပါလက်ပါ စေတနာထက်ထက်သန်သန်နဲ့ ရှင်းပြနေတာ။ ဒီလိုဆရာမျိုးနဲ့ဆို သင်ချင်စိတ်ဖြစ်လာမှာပါ။ သူ့ ရဲ့ PY4E website မှာ Python နဲ့ ပတ်သက်တာတွေ စုပေးထားသလို သူ့စာအုပ်တောင် အလကား ယူလို့ရပါတယ်။ သူ့ရဲ့ Python for Everybody Specialization ဟာ အအောင်မြင်ဆုံး online သင်တန်းပါ။ Coursera၊ edX နဲ့ freeCodeCamp မှာ သင်လို့ရပြီး၊ မစ်ရှီဂန် တက္ကသိုလ်‌ကျောင်းသားတွေအတွက် အလကားပေးထားပါတယ်။ သူ့ course အစ အဆုံး ၁၄ နာရီကြာ ဗီဒီယိုကို ယူကျု မှာလည်း ကြည့်လို့ရပါတယ်။ certificate လက်မှတ်ရဖို့ အတွက်ဆိုရင်တော့ Coursera နဲ့ edX မှာပဲ သင်လိုက်ရင် အဆင်ပြေပါတယ်။

Coursera

Coursera ဟာ online သင်ကြားရေး စနစ်ဖြစ်ပါတယ်။ တက္ကသိုလ်၊ ကုမ္ပဏီ စုစုပေါင်း ၂၇၅ ခု နဲ့ ပူးပေါင်းပြီး course တွေ specialization တွေ ဘွဲ့ degree လက်မှတ်တွေ အတွက် online ကနေ သင်ယူလို့ရတဲ့ နေရာဖြစ်ပါတယ်။ လက်မှတ်ရဖို့ ပညာသင်ထောက်ပံ့ကြေးကို ၁၀၀% အပြည့်အဝ မပေးတော့ပေမယ့် သင်တန်းတွေကိုတော့ အလကားတက်လို့ ရနေပါသေးတယ်။ Python for Everybody Specialization သင်တန်းဟာ Coursera မှာ အအောင်မြင်ဆုံး သင်တန်းထဲက တခုဖြစ်ပါတယ်။ စာသင်သူ/သား ၁.၇ သန်း စာရင်းသွင်းထားနှင့်ပြီးတာ တွေ့ရမှာပါ။

Jovian

Jovian ဟာ လည်း online သင်ကြားရေး စနစ်ဖြစ်တယ်။ Python စလေ့လာသူတွေအတွက် လက်တွေ့ ဦးစားပေး ပုံစံနဲ့သင်ရမှာဖြစ်တယ်။ ‘Data Analysis with Python: Zero to Pandas’ course မှာ data analysis လုပ်ဖို့ Python ၊ Pandas နဲ့ Numpy တွေဘယ်လို သုံးရမလဲ သင်ရမှာဖြစ်ပါတယ်။ သူ့ course အစ အဆုံး ၁၀ နာရီကြာ ဗီဒီယိုကို ယူကျု မှာလည်း ကြည့်လို့ရပါတယ်။

Jupyter Notebook, Google Colab, or Kaggle တွေသုံးရမှာ ဖြစ်ပြီး လေ့ကျင့်ခန်းတွေ (အပြည့်လုပ်ရင် တော်တော်) လုပ်ရမှာဖြစ်ပါတယ်။ လက်မှတ်ရဖို့အတွက် ဆိုရင်တော့ ပိုက်ဆံသွင်းပြီး assignments နဲ့ projects တွေ ပြီးအောင်လုပ်ရမှာဖြစ်ပါတယ်။ ဒါပေမယ့် လေ့လာမယ်ဆိုရင်တော့ အလကားလေ့လာလို့ ရပါတယ်။

freeCodeCamp

freeCodeCamp ကတော့ အကုန် အလကားရတဲ့ နေရာပါ။ စာသင်ဖို့၊ projects လုပ်ဖို့၊ လက်မှတ်.. အကုန် အလကားရပါတယ်။ Python စသင်ဖို့ အဆင်ပြေတဲ့ courses တွေရှိသလို အခြား courses တွေလည်းရှိပါတယ်။ ကွန်ပျူတာသင်ဖို့ အင်္ဂလိပ်စာအခြေခံ သင်္ချာအခြေခံ စသဖြင့်တွေတွေ့ပါတယ်။ ကိုယ်နဲ့ အဆင် ပြေမပြေ ကြည့်ကြည့်ပါ။

freeCodeCamp ဟာ အမေရိကန်အစိုးရကို တရားဝင် မှတ်ပုံတင်ထားတဲ့ အလှူငွေအထောက်အပံ့နဲ့ လည်ပတ်နေတဲ့ အခွန်ကင်းလွတ်ခွင့်ရ ပရဟိတအဖွဲ့အစည်းဖြစ်ပါတယ်။ (United States Federal Tax Identification Number: 82-0779546) လူတိုင်း coding အခမဲ့ ‌သင်ယူနိုင်ဖို့ ရည်ရွယ်ချက်နဲ့ တည်ထောင်ထားတဲ့ အဖွဲ့အစည်းပါ။ သဒ္ဓါတရားထက်သန်သွားပြီလား? အဲ့လိုဆိုရင်တော့ သူ့စာမျက်နှာကတဆင့် သူ့ဝန်ထမ်းတွေ ဝန်ဆောင်မှုတွေ ကုန်ကျစားရိတ်တွေ အတွက် ကူညီ ထောက်ပံ့ပေးလို့ရပါတယ်။

SQLBot

Programming သင်ပြီဆိုတာနဲ့ database language ဖြစ်တဲ့ SQL ဘာသာကို သင်ဖို့ လိုပါတယ်။ S-Q-L တချို့ကလည်း sequel လို့ ခေါ် ကြပါတယ်။ SQLBot ကတော့ သင်ခန်းစာတိုတိုလေးတွေနဲ့ (interactive) အမှန်အမှားချက်ချင်းသိနိုင်တဲ့ လေ့ကျင့်ခန်းလေးတွေ လေ့ကျင့်လို့ရတဲ့နေရဖြစ်ပါတယ်။ SQL စ လေ့လာသူအတွက် အထောက်ကူဖြစ်မှာပါ။ SQLite database ဟာ Python နဲ့ အတူတူပါတဲ့ database ပါ။ Python သွင်းပြီးတာနဲ့ database သုံးလို့ရပါပြီ။ အဆင်ပြေတယ်.. ဟုတ်?

ကဲ ဘယ်လိုလဲ? Python သင်ချင်စိတ် ပေါက်သွားပြီလား?

မှတ်ချက်။ ။ Python အမှတ် တံဆိပ်ကို ဒီစာမျက်နှာမှာ သုံးထားတာက PSF Trademark Usage Policy နဲ့ Python Logo Usage Guidelines တွေနဲ့အညီ သုံးထားတာဖြစ်ပါတယ်။ ပြဿနာတခုခုရှိတယ်ဆိုရင် အီးမေးလ် နဲ့ အကြောင်းကြား ဆက်သွယ်ပေးစေလိုပါတယ်။ Python အမှတ်တံဆိပ် အကြောင်းသိချင်တာဆိုရင်တော့ သူတို့ စာမျက်နှာ မှာ သွားရောက်လေ့လာနိုင်ပါတယ်။
